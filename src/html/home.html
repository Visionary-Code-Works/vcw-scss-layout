Welcome to VCW SCSS Layout

About This Project

This project demonstrates how to structure and compile SCSS files using Webpack, along with integrating Bootstrap and Normalize.css.

Get Started

Clone the repository and install the dependencies to get started with the development:

git clone https://github.com/Visionary-Code-Works/vcw-scss-layout.git
cd vcw-scss-layout
npm install
